# O431-STM32F407
O431的天空星筑基开发板学习笔记

基于嘉立创天空星筑基开发版进行的学习记录，使用c++进行开发

## 开发环境
使用CMake4.2.3，arm-none-eabi-gcc,arm-none-eabi-g++进行开发，
系统环境为Ubuntu24.04，使用HAL库进行开发，所用ide为clion。


## 学习进度
### 学习进行的代码均在sources文件夹中存放
使用Func_Main对原本程序中的主函数进行替换。
### FPU
对FPU功能进行使能，使其支持单精度浮点数运算。
### UART
对串口输出进行重定向，使其支持输出浮点数。
### I2C
已实现AHT20驱动的编写，可以实现温湿度数据的读取。
